    • National Land Use Project

    National Land Use Project

    NATIONAL LAND USE PROJECT

  • The MIS Data Integrations and management using Enterprise Geospatial Technology
  • Description of Actual Services Provided by Your Staff:
  • In the Tenth Five Year Plan (2002-2007), implementation of land use program was included. So, the project initiated its work to update existing land resources maps, to prepare land zoning data and to prepare profile of district level and to create land use data, to prepare land zoning data and to prepare profile for village Development Committees/ municipalities level. The data was prepared in different fiscal years by out sourcing to different consulting forms. So, there is some lack of consistency in the in spatial data format. There is inconsistency in the attributes. There is some inconsistency in spatial referencing and projection systems. There is very little awareness of the user community about the prepared data. In order to bridge the gap between the NLUP products and user requirement, make consistent seamless data; establish easy access of data to the users by adopting online transaction system the project initiated this Data Integration and Management using Enterprise Geospatial Technology Project by outsourcing to the consulting firm.
